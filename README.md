# 🚀 Santander Bootcamp 2026 - AI Java Back-end

Olá! Bem-vindo(a) ao meu repositório central de estudos do **Santander Bootcamp 2026**, especificamente na trilha **AI Java Back-end**. 

Este programa é uma iniciativa incrível oferecida pela **Santander Open Academy** em parceria com a **DIO (Digital Innovation One)**. Ele foi desenhado para alavancar a carreira de desenvolvedores, unindo o poder e a robustez do ecosistema Java com as inovações da Inteligência Artificial.

## 🎯 Sobre o Repositório

Criei este espaço para guardar todos os meus códigos, exercícios práticos, anotações e projetos desenvolvidos ao longo do curso. É aqui que eu registro a minha evolução técnica, partindo da sintaxe básica até chegar em tópicos super em alta no mercado.

## 📁 O que você vai encontrar aqui?

Os módulos e projetos estão divididos em pastas. Atualmente, temos:

* **[📂 ExercicioBasicaoJava](./ExercicioBasicaoJava)**: Minha prática do módulo de "Fundamentos da Linguagem Java". Construí um sisteminha no console com menus interativos, validações e tratamento de erros (fui um pouco além do básico pra testar meus limites! 😅).

*(Novas pastas e projetos serão adicionados conforme eu for avançando nas aulas!)*

## 🛠️ O que estou aprendendo neste Bootcamp?

A ementa da trilha **AI Java Back-end** é bem completa e moderna, abordando:

- **Linguagem Base**: Java (com foco nas novidades do Java 21)
- **Ecosistema Spring**: Spring Boot, Spring Security
- **Boas Práticas e Arquitetura**: Clean Architecture, Design Patterns
- **Inovação (A Cereja do Bolo)**: Integração com APIs e serviços de Inteligência Artificial, como Google Gemini e Whisper.
